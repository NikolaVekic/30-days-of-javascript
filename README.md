# 30 Days of JavaScript

A collection of 30 small JavaScript projects created to practice DOM manipulation, APIs, animations, UI interactions, and full-stack development.

## Projects

- **Day 01 - [Cards](./Day%2001%20-%20Cards/)** - Expandable image cards with smooth transitions.
- **Day 02 - [Steps](./Day%2002%20-%20Steps/)** - A multi-step progress indicator with previous and next controls.
- **Day 03 - [Rock Paper Scissors](./Day%2003%20-%20Rock%20Paper%20Scissors/)** - Play Rock Paper Scissors against the computer.
- **Day 04 - [Color Generator](./Day%2004%20-%20Color%20Generator/)** - Generate random colors with a button click.
- **Day 05 - [Music Player](./Day%2005%20-%20Music%20Player/)** - A simple audio player with playback and progress controls.
- **Day 06 - [Blurry Loading](./Day%2006%20-%20Blurry%20Loading/)** - A loading counter paired with a fading blur effect.
- **Day 07 - [Password Generator](./Day%2007%20-%20Password%20Generator/)** - Generate customizable random passwords.
- **Day 08 - [Stopwatch](./Day%2008%20-%20Stopwatch/)** - Start, stop, and reset a digital stopwatch.
- **Day 09 - [Split Landing Page](./Day%2009%20-%20Split%20Landing%20Page/)** - An interactive split-screen product landing page.
- **Day 10 - [Calculator App](./Day%2010%20-%20Calculator%20App/)** - A browser-based calculator for basic arithmetic.
- **Day 11 - [Scroll Animation](./Day%2011%20-%20Scroll%20Animation/)** - Reveal content cards as the page scrolls.
- **Day 12 - [Weather App](./Day%2012%20-%20Weather%20App/)** - Search for current weather conditions by city.
- **Day 13 - [Quiz App](./Day%2013%20-%20Quiz%20App/)** - Answer multiple-choice questions and view the final score.
- **Day 14 - [Hidden Widget](./Day%2014%20-%20Hidden%20Widget/)** - Expand and collapse a hidden search input.
- **Day 15 - [QR Code Generator](./Day%2015%20-%20QR%20Code%20Generator/)** - Turn entered text or URLs into QR codes.
- **Day 16 - [Internet Speed](./Day%2016%20-%20Internet%20Speed/)** - Estimate internet download speed.
- **Day 17 - [FAQ Section](./Day%2017%20-%20FAQ%20Section/)** - Expandable FAQ questions and answers.
- **Day 18 - [Background Slider](./Day%2018%20-%20Background%20Slider/)** - Browse images with a synchronized background slider.
- **Day 19 - [Clock](./Day%2019%20-%20Clock/)** - An analog clock with the current date and time.
- **Day 20 - [Ripple Effect](./Day%2020%20-%20Ripple%20Effect/)** - Add a ripple animation to button clicks.
- **Day 21 - [Kinetic Loader](./Day%2021%20-%20Kinetic%20Loader/)** - A CSS-based animated loading indicator.
- **Day 22 - [Incrementing Counter](./Day%2022%20-%20Incrementing%20Counter/)** - Animate statistics from zero to their target values.
- **Day 23 - [Content Loader](./Day%2023%20-%20Content%20Loader/)** - Display a skeleton loading state before content appears.
- **Day 24 - [Temperature Converter](./Day%2024%20-%20Temperature%20Converter/)** - Convert values between temperature units.
- **Day 25 - [Dice Roll](./Day%2025%20-%20Dice%20Roll/)** - Roll a virtual die with an animated result.
- **Day 26 - [Character Counter](./Day%2026%20-%20Character%20Counter/)** - Track entered and remaining text characters.
- **Day 27 - [AI Mood Gradient](./Day%2027%20-%20AI%20Mood%20Gradient/)** - Generate color gradients from mood descriptions using AI.
- **Day 28 - [README Generator](./Day%2028%20-%20README%20Generator/)** - Generate structured project README content using AI.
- **Day 29 - [Discover Music](./Day%2029%20-%20Discover%20Music/)** - Find Spotify tracks from natural-language music prompts.
- **Day 30 - [Room Decor](./Day%2030%20-%20Room%20Decor/)** - Generate AI-powered room redesigns from uploaded photos.
