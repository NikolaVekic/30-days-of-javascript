## Description

README Generator is a simple tool that uses the OpenAI API to help format and write professional README files for projects. It provides a straightforward way to generate clean, structured documentation with minimal effort.

## Features

- OpenAI API integration
- Automated README creation
- Simple setup and usage
- Clean and professional output formatting

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js

## Installation

1. Create a `.env` file and add the following:
   - `OPENAI_API_KEY`
   - `PORT`

2. In the `server` folder, install dependencies:
   - `npm install`

3. Start the server:
   - `npm run dev`

4. Open the HTML file to run the frontend.

## Usage

Use README Generator to format and write a professional README file for any project. After starting the server and opening the frontend, provide your project details and generate a polished README.
